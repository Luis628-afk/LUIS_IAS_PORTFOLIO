Password Strength Checker

A simple front‑end tool that teaches users how to build stronger passwords.

Step 1

Plan the Goal

I started by deciding what I want the app to teach: how to spot weak passwords and how to make them stronger.

  • Show if a password is Weak, Medium, or Strong
  • Explain why the password got that score
  • Suggest simple ways to improve it
Step 2

Design the Interface

Next, I designed a very small UI so users can focus on the concept, not on a complicated layout.

  • One password input field
  • A colored strength bar under the input
  • A short text message with tips
Step 3

Define the Rules

Then I created simple checks in JavaScript to “score” the password:

  • Length: less than 8 characters is considered weak
  • Checks for lowercase, uppercase, numbers, and symbols
  • The more conditions it passes, the higher the score
Step 4

Show Feedback to the User

Finally, I connect the score to the UI so users can see the result instantly.

  • Strength label updates as you type
  • Bar color changes from red → yellow → green
  • Short tips explain how to make the password safer
Result

What I Learned

This project helped me connect front‑end development to real cyber security habits in a simple and visual way.
